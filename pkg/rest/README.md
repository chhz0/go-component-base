1. 主要功能特点：

- 链式调用：提供流畅的API设计
- 多种内容类型支持：JSON、表单数据、文件上传
- 自动重试机制：网络错误时自动重试
- 超时控制：默认30秒超时，可自定义
- 响应处理：支持直接解析JSON到结构体
- 多部分表单：支持文件上传和混合表单数据
- 查询参数处理：自动处理URL编码
- 错误处理：详细的错误上下文信息
- 自定义配置：可设置超时时间、重试次数等
